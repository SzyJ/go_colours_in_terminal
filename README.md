# go_colours_in_terminal

Reference escape codes for terminal apps

To run in terminal:
```
go run main.go
```

Sample output (add the escape character before these codes):
```
HEADER:          033[95m
OKBLUE:          033[94m
OKGREEN:         033[92m
WARNING:         033[93m
FAIL:            033[91m
ENDC:            033[0m
BOLD:            033[1m
UNDERLINE:       033[4m
```
